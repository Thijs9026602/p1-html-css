Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Sommige browsers gebruiken de broncode iets anders waardoor het iets kan verschillen.


Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Google Chrome. Op mijn laptop met 8GB RAM gebruikt het circa 120MB RAM op de hoofdpagina. Het percentage is 6,7%.
